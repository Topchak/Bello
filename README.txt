Склонируйте репозиторий с помощью Git:

git clone https://github.com/Topchak/Bello.git

Перейдите в директорию проекта:
cd Bello

Установите зависимости:
npm install

Для сборки проекта с помощью Webpack выполните следующую команду:
npx webpack

Чтобы запустить локальный сервер с фейковым API JSON-Server, используйте команду:
json-server db.json
